# 代码审查清单

## 1. 目的

提供 review 阶段使用的统一检查清单。

## 2. 检查项

| 检查项 ID | 检查内容 | 阻塞级别 |
|-----------|----------|----------|
| REVIEW-001 | 每个 Req ID 是否有代码和测试覆盖。 | blocker |
| REVIEW-002 | 是否存在无关改动或范围蔓延。 | blocker |
| REVIEW-003 | 是否违反模块边界或依赖方向。 | high |
| REVIEW-004 | 是否存在循环查库、事务内外部调用、N+1 查询。 | high |
| REVIEW-005 | 是否存在空 catch、printStackTrace、System.out.println。 | high |
| REVIEW-006 | 是否存在安全、权限、敏感日志风险。 | high |
