# 发布与回滚规范

## 1. 目的

记录发布说明、风险暴露和回滚方案要求。

## 2. 发布说明

| 规则 ID | 内容 | 说明 |
|---------|------|------|
| REL-001 | 本次变更内容。 | 面向交付和审查。 |
| REL-002 | 验证命令和结果。 | 引用测试结果文档。 |
| REL-003 | 已知风险和未解决问题。 | P2 或环境限制必须说明。 |

## 3. 回滚方案

| 规则 ID | 场景 | 必须内容 |
|---------|------|----------|
| RB-001 | strict 变更。 | 回滚步骤、数据回滚、灰度或兼容策略。 |
| RB-002 | 数据库变更。 | 回滚 SQL 或不可回滚说明。 |
| RB-003 | 外部集成变更。 | 降级开关或恢复旧逻辑方案。 |
