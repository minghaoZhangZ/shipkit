# Agent Permission Model

## Analyst / Designer / Planner

允许：读取需求、代码和配置，写入对应 `<change-dir>/ai` 文档。
禁止：修改业务代码，擅自新增依赖。

## Reviewer

允许：只读检查代码、diff、测试结果，写入审查报告。
禁止：修改业务代码，为了通过审查而直接修代码。

## Resolver

允许：修复明确的构建、测试、lint 或审查问题，在实施计划允许范围内最小修改。
禁止：顺手重构，降低质量门禁，修改实施计划禁止的文件。

## Builder

允许：用户确认实施计划后编码。
禁止：编码前绕过实施计划和测试计划，擅自扩大范围。
